# Product Images - Place Your Photos Here

## How to add images:
1. Save your product photos in this folder
2. Name them clearly, e.g.: `pump-casing.jpg`
3. Open `config.js` and update the `image` path for each product

## Recommended sizes:
- Product images: 800 x 600px (or larger, keep under 500KB each)
- Hero/About images: 1920 x 1080px

## Suggested image list:

- about.jpg      (company / factory photo)
